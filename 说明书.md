# 股票分析系统

本系统用于股票数据的抓取、分析和策略回测、AI分析，提供完整的股票分析流程。

## 执行顺序

1. **数据检查与更新**
   - `check_data_updates.py` - 检查数据文件是否是最新日期以及是否有数据文件缺失，自动更新缺失数据
     - `python check_data_updates.py --ticker stock_code` - 执行单只股票的历史数据抓取
     - `python check_data_updates.py` - 执行数据检查与更新
2. **数据抓取**
   - `data_collector.py` - 抓取历史数据
     - `python data_collector.py --ticker stock_code` - 执行单只股票的历史数据抓取
     - `python data_collector.py` - 批量抓取config中股票列表中所有股票的历史数据
   - `stock_company_info_collector.py` - 抓取股票公司信息，包括基础信息、财务信息、行业信息等
     - `python stock_company_info_collector.py --ticker stock_code` - 执行单只股票的公司信息抓取
     - `python stock_company_info_collector.py` - 批量抓取config中股票列表中所有股票的公司信息
   - `stock_market_data_collector.py` - 抓取股票市场数据，包括资金流入流程、融资融券数据等
     - `python stock_market_data_collector.py --ticker stock_code` - 执行单只股票的市场数据抓取
     - `python stock_market_data_collector.py` - 批量抓取config中股票列表中所有股票的市场数据
3. **数据基础分析**
   - `daily/batch_analysis.py` - 批量分析数据质量检查和可视化分析
     - `python daily/batch_analysis.py --ticker stock_code` - 执行单只股票的数据分析
     - `python daily/batch_analysis.py` - 批量分析config中股票列表中所有股票的数据
4. **AI分项信息分析**
   - `batch_analyze_all.py` - 批量分项信息分析
     - `python batch_analyze_all.py --ticker stock_code` - 执行单只股票的数据分析
     - `python batch_analyze_all.py` - 批量分析config中股票列表中所有股票的数据分析
5. **AI综合分析**
   - `stock_ai_comprehensive_analyzer.py` - 综合分析股票数据
     - `python stock_ai_comprehensive_analyzer.py --ticker stock_code` - 执行单只股票的综合分析
     - `python stock_ai_comprehensive_analyzer.py` - 批量分析config中股票列表中所有股票的综合分析

## 文件功能说明

### 1. 数据检查与更新文件

#### check\_data\_updates.py

- **功能**：检查股票数据文件是否是最新日期以及是否有数据文件缺失，自动更新缺失数据
- **使用方法**：
  ```bash
  # 检查单只股票
  python check_data_updates.py --ticker 600313.SH

  # 检查所有股票
  python check_data_updates.py
  ```

## 配置文件

### config.py

- **功能**：系统配置文件，包含股票列表、AI配置、数据目录等配置信息
- **使用方法**：修改文件中的配置项以适应不同的需求

## 数据目录结构

- `data/` - 存储所有股票数据
  - `{stock_code}/` - 每个股票的独立目录
    - `{stock_code}_qfq.csv` - 前复权数据
    - `{stock_code}_indicators.csv` - 技术指标数据
    - `{stock_code}_fund_flow.csv` - 资金流数据
    - `{stock_code}_margin_data.csv` - 融资融券数据
    - `{stock_code}_valuation.csv` - 估值数据
    - `{stock_code}_company_info.json` - 公司信息
    - `{stock_code}_technical_trend_analysis_*.json` - 技术趋势分析报告
    - `{stock_code}_comprehensive_analysis_*.md` - 综合分析报告
    - `{stock_code}_prompt_info_*.md` - 提示词信息

## 使用流程示例

1. **抓取数据**
   ```bash
   # 抓取单只股票数据
   python data_collector.py --ticker 600313.SH
   python stock_company_info_collector.py --ticker 600313.SH
   python stock_market_data_collector.py --ticker 600313.SH

   # 批量抓取所有股票数据
   python data_collector.py
   python stock_company_info_collector.py
   python stock_market_data_collector.py
   ```
2. **基础分析**
   ```bash
   # 分析单只股票
   python daily/batch_analysis.py --ticker 600313.SH

   # 分析所有股票
   python daily/batch_analysis.py
   ```
3. **分项分析**
   ```bash
   # 分析单只股票
   python batch_analyze_all.py --ticker 600313.SH

   # 分析所有股票
   python batch_analyze_all.py
   ```
4. **综合分析**
   ```bash
   # 分析单只股票
   python stock_ai_comprehensive_analyzer.py --ticker 600313.SH

   # 分析所有股票
   python stock_ai_comprehensive_analyzer.py
   ```

## 注意事项

1. 确保网络连接正常，数据抓取需要网络访问
2. 首次运行时，数据抓取可能需要较长时间
3. 分析结果会保存在对应股票的数据目录中
4. 可以通过修改config.py中的股票列表来分析不同的股票
5. 技术趋势分析需要先运行analyze\_technical\_trend.py生成趋势文件
