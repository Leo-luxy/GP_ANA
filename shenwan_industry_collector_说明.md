# shenwan_industry_collector.py 程序说明

## 1. 数据源

该程序从以下数据源获取申万行业信息：

| 数据源 | 描述 | 使用的函数/API |
|-------|------|--------------|
| akshare | 主要数据来源，提供申万行业分类信息 | `ak.sw_index_first_info`、`ak.sw_index_second_info`、`ak.sw_index_third_info`、`ak.sw_index_third_cons` |

## 2. 抓取的数据

程序抓取以下数据：

### 2.1 申万一级行业信息
- 一级行业代码、行业名称、成份个数、静态市盈率、TTM(滚动)市盈率、市净率、静态股息率

### 2.2 申万二级行业信息
- 二级行业代码、行业名称、上级行业、成份个数、静态市盈率、TTM(滚动)市盈率、市净率、静态股息率

### 2.3 申万三级行业信息
- 三级行业代码、行业名称、上级行业、成份个数、静态市盈率、TTM(滚动)市盈率、市净率、静态股息率

### 2.4 行业成分股信息
- 当指定股票代码时，获取该股票所属三级行业的成分股列表，包括股票代码、股票简称、纳入时间、市值、市盈率、市净率、股息率、营收增长率、净利润增长率等

## 3. 保存的文件

程序将数据保存到以下文件中：

| 文件名 | 保存路径 | 数据内容 |
|-------|---------|---------|
| `shenwan_industry_level1.csv` | `{项目根目录}/data/shenwan_industry/` | 申万一级行业信息 |
| `shenwan_industry_level2.csv` | `{项目根目录}/data/shenwan_industry/` | 申万二级行业信息 |
| `shenwan_industry_level3.csv` | `{项目根目录}/data/shenwan_industry/` | 申万三级行业信息 |
| `shenwan_industry_level3_{industry_code}_stocks.csv` | `{项目根目录}/data/shenwan_industry/` | 指定三级行业的成分股信息 |
| `{ticker}_industry_info.json` | `{项目根目录}/data/{ticker}/` | 指定股票的行业信息和行业平均数据 |

其中，`{ticker}` 是股票代码（如 002384.SZ），`{industry_code}` 是行业代码（如 850822.SI）。

## 4. 数据处理特点

1. **数据完整性**：同时获取申万一级、二级、三级行业信息，提供完整的行业分类体系
2. **行业平均计算**：计算行业成分股的平均市值、市盈率、市净率、股息率、增长率等指标
3. **单位标准化**：为行业平均数据添加单位（亿元、%、元），提高数据可读性
4. **精度控制**：行业平均数据保留3位小数，提高数据精度
5. **时间戳管理**：根据当前时间自动计算数据来源时间（17:00之前为前一天，17:00之后为当天）
6. **错误处理**：对每个数据获取步骤都进行了异常捕获，确保程序稳定运行
7. **目录自动创建**：自动创建数据存储目录，确保数据保存路径存在

## 5. 使用方法

1. **处理指定股票**：
   ```bash
   python shenwan_industry_collector.py --ticker 002384.SZ
   ```
   该命令会获取申万一、二、三级行业信息，并为指定股票获取所属行业的详细信息和行业平均数据。

2. **仅获取行业信息**：
   ```bash
   python shenwan_industry_collector.py
   ```
   该命令会获取申万一、二、三级行业信息，并获取部分三级行业的成分股列表作为示例。

## 6. 代码结构

- **get_shenwan_industry_level1 函数**：获取申万一级行业信息并保存到本地
- **get_shenwan_industry_level2 函数**：获取申万二级行业信息并保存到本地
- **get_shenwan_industry_level3 函数**：获取申万三级行业信息并保存到本地
- **get_shenwan_industry_level3_stocks 函数**：获取指定三级行业的成分股列表并保存到本地
- **get_industry_for_stock 函数**：从公司基本信息文件中提取股票的板块名称层级
- **calculate_industry_average 函数**：根据成分股信息计算行业平均数据，添加单位并保留3位小数
- **main 函数**：主函数，处理命令行参数并调用各个数据获取函数

该程序设计合理，结构清晰，能够有效地从 akshare 获取申万行业信息，并将数据保存到相应的文件中，为后续的分析和处理提供了便利。
