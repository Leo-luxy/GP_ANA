# 阈值配置系统使用说明

## 📖 快速开始

**不需要自己填数字！** 直接选择预设的配置即可。

### 方式一：自动选择（推荐）
程序会根据股票市值自动选择合适的配置。

### 方式二：按风格选择
- **保守型**：更谨慎，要求更高，适合风险厌恶者
- **平衡型**：适中，日常使用推荐
- **激进型**：更宽容，容忍度更高

---

## 📊 预设配置一览

| 配置名称 | 适用范围 | 特点 |
|---------|---------|------|
| **LARGE_CAP** | 市值 > 1000亿 | 大盘蓝筹股，研报更多，标准更严 |
| **MID_CAP** | 市值 100-1000亿 | 中盘成长股，标准适中 |
| **SMALL_CAP** | 市值 < 100亿 | 小盘股，研报较少，标准更宽松 |
| **CONSERVATIVE** | 所有股票 | 保守风格，要求更高 |
| **BALANCED** | 所有股票 | 平衡风格，推荐日常使用 |
| **AGGRESSIVE** | 所有股票 | 激进风格，更宽容 |

---

## 💡 每个配置是什么意思？

### 研报相关阈值
| 配置项 | 作用 |
|-------|------|
| overweight_strong = 90 | 买入+增持评级占90%以上认为机构**一致看好** |
| overweight_good = 70 | 买入+增持评级占70%以上认为机构**普遍看好** |
| sell_alert = 30 | 减持+卖出评级占30%以上认为机构**一致看空** |
| growth_strong = 30 | 盈利预期增长30%以上认为**增长强劲** |
| growth_good = 15 | 盈利预期增长15%以上认为**增长良好** |
| coverage_high = 20 | 近3个月研报20篇以上认为**机构高度关注** |
| coverage_medium = 10 | 近3个月研报10篇以上认为**机构关注较高** |
| coverage_low = 5 | 近3个月研报少于5篇认为**机构关注度低** |

### 股东相关阈值
| 配置项 | 作用 |
|-------|------|
| top1_dominant = 50 | 第一大股东持股50%以上认为**一股独大** |
| inst_hold_high = 15 | 机构持仓15%以上认为**机构持仓较高** |
| num_change_high = 10 | 股东数变化10万以上认为**有显著变化** |
| etf_passive = 4 | ETF持仓4家以上认为**被动资金主导** |

### 情绪相关阈值
| 配置项 | 作用 |
|-------|------|
| main_flow = 5 | 主力资金净流入净占比5%以上认为**显著流入** |
| margin_change = 10 | 融资余额变化10%以上认为**资金活跃** |

### 财务相关阈值
| 配置项 | 作用 |
|-------|------|
| roe_excellent = 15 | ROE > 15%认为**盈利能力优秀** |
| roe_good = 10 | ROE > 10%认为**盈利能力良好** |
| roe_normal = 5 | ROE > 5%认为**盈利能力一般** |
| debt_ratio_alert = 70 | 资产负债率 > 70%认为**杠杆偏高** |

## 🚀 如何在代码中使用

### 查看配置指南
```bash
cd /path/to/GP_ANA
python config/thresholds_config.py
```

### 自动选择配置
```python
from config.thresholds_config import get_config_by_market_cap

# 根据市值选择（单位：亿元）
config = get_config_by_market_cap(1500)  # 1500亿，会选择 LARGE_CAP
print(config["name"])  # 输出: 大盘股配置
```

### 按风格选择配置
```python
from config.thresholds_config import get_config_by_style

config = get_config_by_style("conservative")  # 选择保守型
config = get_config_by_style("balanced")       # 选择平衡型
config = get_config_by_style("aggressive")     # 选择激进型
```

### 列出所有配置
```python
from config.thresholds_config import list_all_configs

all_configs = list_all_configs()
for cfg in all_configs:
    print(f"{cfg['key']}: {cfg['name']} - {cfg['description']}")
```

## 📝 常见问题

### Q: 我不懂投资，用哪个配置？
A: **直接用 BALANCED（平衡型）**，这是最通用的配置。

### Q: 什么是 HHI 指数？
A: HHI（赫芬达尔-赫希曼指数）是衡量股权集中度的国际标准：
- HHI > 2500：高度集中
- 1800 < HHI < 2500：较高集中
- 1000 < HHI < 1800：中度集中
- HHI < 1000：高度分散

这个阈值是**美国司法部反垄断标准**，完全通用，不需要调整。

### Q: 什么是 PEG 指标？
A: PEG（市盈率相对盈利增长比率）是彼得林奇提出的经典估值方法：
- PEG < 1：估值偏低
- 1 < PEG < 2：估值适中
- PEG > 2：估值偏高

这个阈值是**投资经典标准**，完全通用，不需要调整。

### Q: 这些阈值是谁设定的？靠谱吗？
A: 大部分阈值都来自：
1. **国际通用标准**（HHI、PEG、流动/速动比率）
2. **财务分析经典标准**（ROE、利息保障倍数）
3. **国内股市常见实践**（机构持仓、研报覆盖）

所有阈值都经过验证，可以放心使用。
