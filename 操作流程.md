# 股票分析操作流程

## 一、新股票初始化流程

### 1. 数据抓取阶段

#### 第一步：抓取低频数据
```bash
# 抓取基本信息、财务数据、股东数据等低频更新数据
python check_periodic_data_updates.py --ticker 股票代码

# 例如：
python check_periodic_data_updates.py --ticker 300433.SZ
```

**抓取的数据包括**：
- 公司基本信息（月度更新）
- 财务指标（季度更新）
- 东方财富财务数据（季度更新）
- 股东数据（季度更新）
- 股东户数（月度更新）
- 重要缺失数据（除权除息、龙虎榜等）（月度更新）
- 北向资金数据（月度更新）

#### 第二步：抓取日更数据
```bash
# 抓取价格、资金流、融资融券、估值等日更数据
python check_data_updates.py --ticker 股票代码

# 例如：
python check_data_updates.py --ticker 300433.SZ
```

**抓取的数据包括**：
- 前复权价格数据
- 资金流数据
- 融资融券数据
- 估值数据
- 技术指标数据

### 2. 数据分析阶段

#### 第三步：分析日更数据
```bash
# 分析资金流、融资融券、估值、技术趋势等日更数据
python batch_analyze_daily.py --ticker 股票代码

# 例如：
python batch_analyze_daily.py --ticker 300433.SZ
```

**分析的内容**：
- 资金流分析
- 融资融券分析
- 估值分析
- 技术趋势分析

#### 第四步：分析低频数据
```bash
# 分析财务报表、研究报告、股东结构等低频数据
python batch_analyze_periodic.py --ticker 股票代码

# 例如：
python batch_analyze_periodic.py --ticker 300433.SZ
```

**分析的内容**：
- 财务报表分析
- 研究报告分析
- 股东结构分析
- 东方财富财务数据分析
- 业绩预告与分红分析

### 3. 综合分析阶段

#### 第五步：生成综合分析报告
```bash
# 整合所有分析结果，生成综合分析报告
python stock_ai_comprehensive_analyzer.py --ticker 股票代码

# 例如：
python stock_ai_comprehensive_analyzer.py --ticker 300433.SZ
```

**生成的报告**：
- 综合所有分析维度
- 提供投资建议
- 风险评估
- 投资策略建议

## 二、每日更新流程

### 1. 数据更新

#### 第一步：更新日更数据
```bash
# 只更新日更数据，低频数据无需每日更新
python check_data_updates.py --ticker 股票代码

# 例如：
python check_data_updates.py --ticker 300433.SZ
```

### 2. 数据分析

#### 第二步：分析最新日更数据
```bash
# 分析最新的日更数据
python batch_analyze_daily.py --ticker 股票代码

# 例如：
python batch_analyze_daily.py --ticker 300433.SZ
```

### 3. 综合分析

#### 第三步：生成最新综合分析报告
```bash
# 基于最新数据生成综合分析报告
python stock_ai_comprehensive_analyzer.py --ticker 股票代码

# 例如：
python stock_ai_comprehensive_analyzer.py --ticker 300433.SZ
```

## 三、定期更新流程

### 每月或每季执行一次

```bash
# 1. 更新低频数据
python check_periodic_data_updates.py --ticker 股票代码

# 2. 分析低频数据
python batch_analyze_periodic.py --ticker 股票代码

# 3. 更新日更数据
python check_data_updates.py --ticker 股票代码

# 4. 分析日更数据
python batch_analyze_daily.py --ticker 股票代码

# 5. 生成综合分析报告
python stock_ai_comprehensive_analyzer.py --ticker 股票代码
```

## 四、批量处理多个股票

### 初始化多个股票
```bash

python check_periodic_data_updates.py
python check_data_updates.py
python batch_analyze_daily.py
python batch_analyze_periodic.py
python stock_ai_comprehensive_analyzer.py
```

### 每日更新多个股票
```bash
# 批量处理config.py中的所有股票
python check_data_updates.py
python batch_analyze_daily.py
python stock_ai_comprehensive_analyzer.py
```

## 七、注意事项

1. **数据抓取**：首次运行时可能需要较长时间，因为要抓取大量历史数据
2. **API限制**：部分数据抓取可能受到API调用频率限制，程序已内置重试机制
3. **增量更新**：所有数据抓取程序都支持增量更新，只获取新数据
4. **报告覆盖**：相同日期的分析报告会覆盖原来的文件
5. **错误处理**：程序包含错误处理机制，单个数据抓取失败不会影响整体流程

## 八、故障排除

- **数据抓取失败**：检查网络连接，API是否可用
- **分析报告生成失败**：检查数据文件是否存在，格式是否正确
- **Ollama AI调用失败**：检查Ollama服务是否运行，模型是否正确配置
- **程序执行错误**：检查Python环境，依赖包是否安装完整
