# analyze_margin_data.py 设计说明

## 1. 数据文件分析

`analyze_margin_data.py` 引用了以下数据文件，每个文件都有其特定的含义和作用：

### 1.1 `{ticker}_margin_data.csv`
- **含义**：融资融券数据，包含公司的融资买入额、融资余额、融券卖出量、融券余量、融券余额、融资融券余额等指标
- **作用**：分析市场对该股票的多空情绪、资金流向和市场活跃度
- **来源**：由相关数据采集脚本采集

### 1.2 `{ticker}_company_basic.json`
- **含义**：公司基本信息数据，包含公司的基本信息、经营范围、规模比较等数据
- **作用**：提供公司的基本背景信息，帮助分析融资融券数据的上下文
- **来源**：由 `stock_company_info_collector.py` 采集

### 1.3 `{ticker}_qfq.csv`
- **含义**：前复权交易数据，包含股票的收盘价、开盘价、最高价、最低价、成交额、成交量、换手率等指标
- **作用**：提供股票的交易数据，计算涨跌幅和振幅，与融资融券数据结合分析
- **来源**：由相关数据采集脚本采集

### 1.4 `{ticker}_valuation.csv`
- **含义**：估值数据，包含公司的PE(TTM)、PE(静)、市净率、PEG值、市现率、市销率、总市值、流通市值等指标
- **作用**：提供公司的估值数据，帮助评估融资融券数据的合理性和投资价值
- **来源**：由相关数据采集脚本采集

## 2. 量化分析角度评估

### 2.1 当前数据覆盖情况

从量化分析角度来看，当前数据已经相当全面地覆盖了融资融券分析的主要维度：

#### 2.1.1 多空情绪分析
- 融资余额与融券余额比例分析
- 融资余额变化趋势分析
- 融券余额变化趋势分析

#### 2.1.2 资金流向分析
- 融资买入额变化趋势分析
- 融资偿还额变化趋势分析
- 融券卖出量变化趋势分析
- 融券偿还量变化趋势分析

#### 2.1.3 市场活跃度分析
- 融资融券余额与流通市值比例分析
- 融资买入额与成交额比例分析
- 换手率变化趋势分析

#### 2.1.4 价格关联分析
- 融资余额变化与股价涨跌幅相关性分析
- 前一日融资买入额占比与当日涨跌幅相关性分析

#### 2.1.5 估值分析
- PE(TTM)、PE(静)分析
- 市净率、PEG值分析
- 市现率、市销率分析
- 总市值、流通市值分析

## 3. 结论

当前的数据文件已经相当全面地覆盖了融资融券分析的主要维度，能够满足大多数量化分析的需求。通过这些数据，可以进行深入的多空情绪分析、资金流向分析、市场活跃度分析、价格关联分析和估值分析，为投资决策提供有力支持。

特别是，通过以下改进，分析报告的质量和深度得到了显著提升：

1. **数据预处理增强**：计算融资偿还额和融券偿还量，提供更完整的资金流动数据
2. **交易数据整合**：整合前复权交易数据，计算涨跌幅和振幅，与融资融券数据结合分析
3. **估值数据集成**：添加估值数据，帮助评估融资融券数据的合理性和投资价值
4. **相关性分析**：分析融资融券数据与股价变动的关联关系，提供量化指标
5. **单位标准化**：统一资金数据单位为万元，股数数据单位为股，提高数据可读性
6. **数据排序一致性**：确保数据按日期升序排列，提高分析的准确性

这些改进使得分析报告能够更全面地反映市场对股票的多空情绪和资金流向，为投资决策提供更有力的支持。

## 4. 代码结构设计

### 4.1 核心功能模块

1. **数据加载模块**：负责加载融资融券数据、公司基本信息、交易数据和估值数据
2. **数据预处理模块**：计算融资偿还额和融券偿还量，处理数据单位和格式
3. **数据提取模块**：提取融资融券相关数据和统计指标
4. **提示词构建模块**：构建AI分析提示词，包含所有必要的数据
5. **AI分析模块**：调用本地Ollama AI进行深度融资融券分析
6. **结果保存模块**：保存提示词和分析结果到对应股票文件夹

### 4.2 数据处理流程

1. 加载融资融券数据文件
2. 提取融资融券相关数据，计算融资偿还额和融券偿还量
3. 加载公司基本信息文件，获取公司基本信息和估值数据
4. 加载交易数据文件，获取最近20日的交易数据
5. 构建AI分析提示词，包含公司基本信息、交易数据、估值数据、融资融券统计数据和最近20日的融资融券数据
6. 获取AI分析结果
7. 保存提示词和分析结果到对应股票文件夹

### 4.3 关键技术特点

1. **多维度融资融券分析**：从多空情绪、资金流向、市场活跃度、价格关联和估值等多个维度进行分析
2. **数据预处理**：计算融资偿还额和融券偿还量，提供更完整的资金流动数据
3. **单位标准化**：统一资金数据单位为万元，股数数据单位为股，提高数据可读性
4. **数据排序一致性**：确保数据按日期升序排列，提高分析的准确性
5. **AI辅助分析**：利用本地Ollama AI进行深度融资融券分析，提供专业的投资建议
6. **结构化报告**：生成结构化的分析报告，便于阅读和理解
7. **结果保存**：将提示词和分析结果保存到对应股票文件夹，便于后续查看和使用
8. **数据完整性检查**：检查数据文件是否存在，确保分析的可靠性

## 5. 应用场景

1. **投资决策支持**：为投资者提供全面的融资融券分析，辅助投资决策
2. **市场情绪监控**：监控市场对股票的多空情绪，及时发现市场热点和风险
3. **资金流向分析**：分析资金流向，识别主力资金的动向
4. **短期走势预测**：基于融资融券数据和价格关联分析，预测股票的短期走势
5. **风险评估**：评估融资融券数据带来的风险，如杠杆踩踏风险、估值回归风险等
6. **投资策略制定**：基于融资融券分析，制定投资策略
7. **投资组合管理**：评估投资组合中各股票的融资融券状况，优化投资组合
