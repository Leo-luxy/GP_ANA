# calculate_financial_indicators.py 程序说明

## 1. 功能概述

该程序的主要功能是：**从财务指标JSON文件中读取数据，计算量化分析可能用到的财务指标，并保存计算结果**。

具体来说，程序会：
- 读取指定股票的财务指标JSON文件
- 计算五大类财务指标：盈利能力、偿债能力、运营能力、现金流和成长能力
- 将计算结果保存到新的JSON文件中，包含计算的指标值和使用的原始数据

## 3. 计算的财务指标

### 3.1 盈利能力指标

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 毛利率 | (营业收入 - 营业成本) / 营业收入 × 100% | 反映企业产品的盈利能力 |
| 净利率 | 净利润 / 营业收入 × 100% | 反映企业整体盈利能力 |
| 营业利润率 | 营业利润 / 营业收入 × 100% | 反映企业经营活动的盈利能力 |
| EBITDA | 营业利润 + 财务费用的绝对值 | 近似计算，缺少折旧摊销数据 |
| 基本每股收益 | 直接从数据中获取 | 反映每股盈利能力 |
| 稀释每股收益 | 直接从数据中获取 | 考虑稀释效应后的每股收益 |
| 营收同比增长率 | (当期营业收入 - 去年同期营业收入) / 去年同期营业收入 × 100% | 反映营业收入的增长情况 |
| 净利润同比增长率 | (当期净利润 - 去年同期净利润) / 去年同期净利润 × 100% | 反映净利润的增长情况 |

### 3.2 偿债能力指标

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 资产负债率 | 负债合计 / 资产总计 × 100% | 反映企业的负债水平 |
| 流动比率 | 流动资产合计 / 流动负债合计 | 反映企业短期偿债能力 |
| 速动比率 | (流动资产合计 - 存货) / 流动负债合计 | 反映企业即时偿债能力 |
| 利息保障倍数 | 营业利润 / 财务费用的绝对值 | 反映企业支付利息的能力 |

### 3.3 运营能力指标

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 总资产周转率 | 营业收入 / 资产总计 | 反映企业资产的利用效率 |
| 应收账款周转率 | 营业收入 / 应收票据及应收账款 | 反映应收账款的周转速度 |
| 存货周转率 | 营业成本 / 存货 | 反映存货的周转速度 |
| 固定资产周转率 | 营业收入 / 固定资产及清理合计 | 反映固定资产的利用效率 |

### 3.4 现金流指标

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 经营活动现金流量净额 | 直接从数据中获取 | 反映企业经营活动的现金流入流出情况 |
| 投资活动现金流量净额 | 直接从数据中获取 | 反映企业投资活动的现金流入流出情况 |
| 筹资活动现金流量净额 | 直接从数据中获取 | 反映企业筹资活动的现金流入流出情况 |
| 现金及现金等价物净增加额 | 直接从数据中获取 | 反映企业现金的净变化情况 |
| 经营活动现金流量净额/净利润 | 经营活动现金流量净额 / 净利润 | 反映企业利润的现金含量 |
| 经营活动现金流量净额/营业收入 | 经营活动现金流量净额 / 营业收入 × 100% | 反映企业营业收入的现金含量 |

### 3.5 成长能力指标

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 营收同比增长率 | (当期营业收入 - 去年同期营业收入) / 去年同期营业收入 × 100% | 反映营业收入的年度增长情况 |
| 净利润同比增长率 | (当期净利润 - 去年同期净利润) / 去年同期净利润 × 100% | 反映净利润的年度增长情况 |
| 营收单季环比增长率 | (当期单季营收 - 上期单季营收) / 上期单季营收 × 100% | 反映营业收入的季度增长情况 |
| 净利润单季环比增长率 | (当期单季净利润 - 上期单季净利润) / 上期单季净利润 × 100% | 反映净利润的季度增长情况 |
| 营收单季同比增长率 | (当期单季营收 - 去年同期单季营收) / 去年同期单季营收 × 100% | 反映单季度营业收入的年度增长情况 |
| 总资产同比增长率 | (当期资产总计 - 去年同期资产总计) / 去年同期资产总计 × 100% | 反映企业规模的增长情况 |
| 净资产同比增长率 | (当期净资产 - 去年同期净资产) / 去年同期净资产 × 100% | 反映企业所有者权益的增长情况 |

## 5. 使用方法

### 5.1 处理指定股票

```bash
python calculate_financial_indicators.py --ticker 300433.SZ
```

### 5.2 处理配置文件中的所有股票

```bash
python calculate_financial_indicators.py
```

其中，股票列表从 config.py 中的 STOCK_TICKERS 获取。

## 6. 代码结构

- **FinancialIndicatorsCalculator 类**：核心计算类，包含以下方法：
  - `_load_data()`：加载JSON数据
  - `_get_profit_data()`：获取利润表数据
  - `_get_balance_data()`：获取资产负债表数据
  - `_get_cash_flow_data()`：获取现金流量表数据
  - `_get_same_month_data()`：获取相同月份的数据
  - `_extract_date_info()`：提取日期信息
  - `calculate_profitability_indicators()`：计算盈利能力指标
  - `calculate_solvency_indicators()`：计算偿债能力指标
  - `calculate_operation_indicators()`：计算运营能力指标
  - `calculate_cash_flow_indicators()`：计算现金流指标
  - `calculate_growth_indicators()`：计算成长能力指标
  - `calculate_all_indicators()`：计算所有指标
  - `save_results()`：保存结果

- **main 函数**：主函数，处理命令行参数并调用 FinancialIndicatorsCalculator 类的方法。

## 7. 数据处理特点

1. **错误处理**：程序会检查输入文件是否存在，避免因文件不存在导致的错误。
2. **数据完整性**：对于每个指标，程序会检查所需数据是否存在，只有在数据完整时才进行计算。
3. **数据溯源**：在保存结果时，程序会记录每个指标使用的原始数据，便于后续分析和验证。
4. **日期处理**：程序会提取报告日的年份和月份，用于计算同比和环比增长率。
5. **结果格式化**：计算结果会保留两位小数，确保数据的一致性和可读性。

该程序设计合理，结构清晰，能够有效地从财务指标JSON文件中计算出各种财务指标，并将结果保存到新的JSON文件中，为后续的量化分析提供了便利。
